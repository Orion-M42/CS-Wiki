## 🔔 解题思路

### 二分查找

每行每行的二分查找


```java
public class Solution {
    public boolean Find(int target, int [][] array) {
        
        for (int i = 0; i < array.length; i ++) {
            int left = 0;
            int right = array[0].length - 1;
            while (left <= right) {
                int mid = left + ((right - left) >> 1);
                if (array[i][mid] == target) {
                    return true;
                }
                else if (array[i][mid] > target) {
                    right = mid - 1;
                }
                else {
                    left = mid + 1;
                }
            }
        }
        
        return false;
    }
}
```

时间复杂度 O(NLogN)

### 线性查找（推荐）

解题思路：利用二维数组行列递增特性

主要思路：

由于行列递增，可以得出：

- **在一列中的某个数字，其上的数字都比它小**
- **在一行中的某个数字，其右的数字都比它大**

搜索流程：

- 首先从数组左下角搜索.
- 如果当前数字大于target,那么查找往上移一位,如果当前数字小于target,那么查找往右移一位。
- 查找到 target,返回true; 如果越界，返回false;

示例如下：

![](https://uploadfiles.nowcoder.com/images/20210623/809820957_1624415093930/CF34A84A75CE743E086BA50AB6363B9E)

时间复杂度 O(M + N)

```java
public class Solution {
    public boolean Find(int target, int [][] array) {
        int curX = array.length - 1;
        int curY = 0;
        
        while (curX >= 0 && curY < array[0].length) {
            int num = array[curX][curY];
            if (num == target) {
                return true;
            }
            // 右移 增大
            else if (num < target) {
                curY ++;
            }
            // 上移 减小
            else {
                curX --;
            }
        }
        
        return false;
    }
}
```
