## 🔔 解题思路

注意这题有空间复杂度的限制，不然可以新开辟一个数组做

三次翻转

- reverse 全部
- reverse 开头 m 个
- reverse 剩余 n - m 个

需要注意 m > n 的情况，用 m % n 来处理


```java
import java.util.*;


public class Solution {
    /**
     * 旋转数组
     * @param n int整型 数组长度
     * @param m int整型 右移距离
     * @param a int整型一维数组 给定数组
     * @return int整型一维数组
     */
    public int[] solve (int n, int m, int[] a) {
        // write code here
        reverse(a, 0, n - 1);
        reverse(a, 0, (m % n) - 1 );
        reverse(a, (m % n) , n - 1);
        
        return a;
    }
    
    // 翻转 a[left, right]
    private void reverse(int[] a, int left, int right) {
        while (left <= right) {
            int temp = a[left];
            a[left] = a[right];
            a[right] = temp;
            
            left ++;
            right --;
        }
    }
}
```
